1.Создать директорию для проекта

2. В данной директории необходимо создать виртуальное окружение
python3 -m venv название окружение например также venv
активировать окружение:
source название_окружение/bin/activate

3.В папке проекта необходимо создать файл зависимостей requirements.txt и прописать внутри все нужные библиотеки например:
    django
    psycopg2-binary
    djangorestframework

4. Установить все зависимости в виртуальное окружение
pip install -r requirements.txt

5.Создаем Django проект:
django-admin startproject название проекта . #!!!!!!!!!точку не забыть!!!!!!!!!!!

6.Создаем приложение проекта
./manage.py startapp название

7.Создаем базу данных для проекта
--- командной строке пишем psql --
----CREATE DATABASE название БД

8. Настройка проекта
-Указываем приложение в INSTALLED_APPS
-Указываем настройки БД
-Указываем настройки языка, часового пояса

9. Создаем модель пользователя
в settings добавляем настройку AUTH_USER_MODEL

10.Определяем остальные модели

11.Делаем миграции
./manage.py makemigrations
./mange.py migrate

12.Создать админа
./manage.py createsuperuser
